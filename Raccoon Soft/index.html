Fabrica de Software

O que é

A Fábrica de Software é uma disciplina que permiter ao aluno colocar em prática tudo o que foi ensinado durante o curso de Sistemas de Informação. Além de simular um ambiente coorporativo

Objetivo

Preparar e adequar os alunos para o mercado de trabalho, os colocando como protagonista em um projeto completo de fábrica de software. Onde a equipe deve ser auto gerenciada para o sucesso do projeto.

Dificuldades

Trabalhar em equipe e saber dividir as tarefas. Tempo para a consepção do projeto é pequeno, visto que muitas ferramentas são necessárias aprender do zero.
